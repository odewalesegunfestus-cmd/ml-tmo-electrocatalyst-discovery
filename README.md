# ML-TMO-Electrocatalyst-Discovery

Machine Learning-guided discovery of multimetal transition metal oxide electrocatalysts for green hydrogen production.

## Project Objective

This project explores the use of machine learning to predict the oxygen evolution reaction (OER) performance of transition metal oxide electrocatalysts.

## Dataset

Catalyst compositions:

- Ni
- Co
- Fe

Target property:

- Overpotential (mV)

## Workflow

1. Load catalyst dataset using Pandas.
2. Visualize catalyst performance.
3. Train a Linear Regression model.
4. Predict virtual catalyst performance.
5. Identify the best catalyst candidate.

## Project Structure

data/ → catalyst dataset

src/ → Python source code

figures/ → generated plots

results/ → future ML outputs
